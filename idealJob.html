Ideal Job - Full Stack Developer

My Description

A full stack web developer knows and works on both the client side and the server side of an I.T project. The reasons why being a fullstack developer appeals to me are; 1 - constant learning; 2 - flexibility of work; and 3 - being able to complete a project solo.

The areas of competence of a full stack web developer are naturally very broad, and due to this, people who perform this work are constantly looking to improve and constantly learning. This ticks a hugely important box that I have in order to be fulfilled and motivated in a work role.

Having a career which is flexible around home life is not an absolute must, but having that trait is attractive in a work role.

Finally, being able to complete a project solo leaves open many doors. It allows one to complete pet projects, or to be entrepreneurial and start on large projects with the hope of monetizing it.

Skills Required

  • Front end - E.g HTML JavaScript, CSS etc
  • Backend - E.g Ruby, Python, PHP
  • Database - E.g SQL

Full stack developers also need to know industry best practices and be aware of best user experiences.

Current Skills

  • Completed Intro to HTML/CSS via Khan Academy
  • Completed Intro to SQL via Khan Academy
